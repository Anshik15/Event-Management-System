Available API Endpoints

Authentication:
POST /api/auth/signup/ - Create new user/vendor
POST /api/auth/login/ - Login
POST /api/auth/logout/ - Logout
GET /api/auth/user/ - Get current user
Admin Endpoints:
GET/POST/PUT/DELETE /api/admin/users/ - Manage users
GET/POST/PUT/DELETE /api/admin/vendors/ - Manage vendors
POST /api/admin/membership/add/ - Add membership
GET/PUT /api/admin/membership/update/ - Update membership
Vendor Endpoints:
GET/POST/DELETE /api/vendor/items/ - Manage items
GET /api/vendor/product-status/ - View product status
User Endpoints:
GET /api/user/vendors/ - List vendors
GET /api/user/items/ - Browse items
GET/POST/PUT/DELETE /api/user/cart/ - Manage cart
POST /api/user/checkout/ - Checkout
GET /api/user/orders/ - View orders
